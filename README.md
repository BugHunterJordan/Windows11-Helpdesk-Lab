# Windows 11 Help Desk Home Lab

## Overview
This home lab simulates common Tier 1 Help Desk tasks performed on a Windows 11 workstation.

## Environment

- Windows 11 Pro (25H2)
- Intel Core i5
- 8 GB RAM
- Local Administrator account
- Standard User account

## Skills Demonstrated

- Windows user account management
- Administrator permissions
- Windows Update
- Software installation and removal
- Network troubleshooting
- DNS verification
- Command Prompt
- Documentation using help desk tickets
- System maintenance

## Lab Tasks

### 1. Created Local User Account

**Issue**
New employee required a Windows account.

**Actions**

- Created Standard User
- Verified login
- Confirmed account functionality

**Result**

Account successfully created.

---

### 2. Administrator Permissions

**Issue**

User needed software installed.

**Actions**

- Temporarily changed account to Administrator
- Installed software
- Returned account to Standard User

**Real-world note**

In production environments, IT staff would typically log in with their own administrator credentials rather than changing the user's account type.
